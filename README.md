# goproxy-shell

goproxy服务端部署脚本

# 版本号及说明
v0.1.0 demo

现有功能：

1.http代理包括 kcp tcp socks 含有或不含有tls加密的部署

2.tls证书生成

3.命令行杂项设置

尚未完成的功能：

1.多级代理

2.非http代理均未完成

3.多proxy任务开机启动

4.tls证书回传

下一步会逐步完善这些功能
